# 우리집 가계부

PC와 모바일에서 모두 사용 가능한 개인 가계부 앱입니다.

## GitHub Pages 배포 방법

1. GitHub에서 새 저장소(repository) 생성
2. 이 폴더의 파일들을 전부 업로드 (또는 git push)
3. 저장소 → **Settings** → **Pages** → Source를 `main` 브랜치 / `/ (root)` 로 설정 → **Save**
4. 잠시 후 `https://[사용자명].github.io/[저장소명]/` 으로 접속

## 기능

| 화면 | 기능 |
|------|------|
| 홈 | 월별 잔액·수입·지출 요약, 주체별 현황, 최근 내역 |
| 내역 | 월별 전체 거래 목록, 입금/출금/주체 필터 |
| 추가 | 거래 입력 (구분·날짜·계정·주체·내용·금액), 항목 탭 수정 |
| 통계 | 월별 막대 차트, 지출/수입 도넛 차트, 주체별 상세 |

## 계정 분류

- **입금**: 개인연금, 보너스, 월급 + 직접 추가 가능
- **출금**: 주식, 금, 개인연금, 대출금상환, 비과세연금, 생활비 + 직접 추가 가능

## 데이터 저장

브라우저의 `localStorage`에 저장됩니다.
같은 기기·브라우저에서는 데이터가 유지되며, 기기 간 동기화는 지원하지 않습니다.
